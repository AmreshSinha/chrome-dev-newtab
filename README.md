# chrome-dev-newtab
A dev friendly new tab for Google Chrome.  

**features** :  
- Show memory and cpu usage ✅
- Google or Stackoverflow search bar ✅
- Easily delete browser data ✅
- Easy access to browser history ✅
- Javascript sandbox ✅
- Lighthouse integration ✅
- Markdown editor ✅
- Light and dark theme ⏳
